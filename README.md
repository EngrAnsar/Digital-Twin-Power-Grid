A **Digital Twin simulation of an electrical power grid** that uses **Artificial Intelligence to predict faults, overloads, and voltage instability**.  
This project demonstrates how **AI + power system simulation** can help modern utilities monitor and optimize grid performance.

---

## 🚀 Features

⚡ Power grid simulation using **Pandapower**  
🤖 **AI-based fault detection** using machine learning  
📊 Real-time **grid monitoring dashboard**  
📉 Load, voltage, and line loading visualization  
🌐 Deployable **Streamlit web app**
